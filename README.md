# Volume Controller using Hand Tracking

A volume controller using hand tracking with Python using `tkinter`, `cv2`, `mediapipe`, and `pycaw` libraries.

## Installation

1. Install the required modules:

   ```bash
   pip install tkinter==0.1.0
   pip install ttkbootstrap==1.10.1
   pip install opencv-python==4.6.0.66
   pip install pillow==9.3.0
   pip install mediapipe==0.10.9
   pip install numpy==1.26.2
   pip install pycaw==20230407
   pip install comtypes==1.2.1



## How does the program work?

- The program will initiate once the user displays five fingers

https://github.com/Abbas-Ghazavi/Volume-Controller-using-Hand-Tracking/assets/118556497/046928a6-bbe5-49f8-ba31-84af179e9fe9


- After launching the program, adjust the volume by using the space between two fingers, approximately 2 inches apart.

https://github.com/Abbas-Ghazavi/Volume-Controller-using-Hand-Tracking/assets/118556497/97a4cee1-2596-41a6-bc5b-4e504e51a255


- To exit the hand detection mode and return to the default state, simply display three fingers.

https://github.com/Abbas-Ghazavi/Volume-Controller-using-Hand-Tracking/assets/118556497/e72849fc-461e-4a1e-9647-c0227ca7c72b
